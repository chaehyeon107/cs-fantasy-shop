위 프로젝트는 전북대학교 ‘웹서비스 설계’ 과목의 Term Project로 진행한 백엔드 실습용 사이드 프로젝트입니다.

‘컴공 판타지 아이템 쇼핑몰’은 컴퓨터공학 세계관을 판타지 RPG 형태로 재해석한 온라인 아이템 상점입니다.
이 서비스는 개발자를 유저로 설정하고, 등급과 스탯에 따라 필요한 컴공 아이템(예: 마법 스택, 알고리즘 스킬북, 메모리 갑옷, 디버깅 포션 등)을 구매할 수 있도록 구성했습니다.

개발자들 사이에서 자주 회자되는 컴공 밈과 장비 메타포를 유쾌하게 반영한 콘텐츠가 특징이며,
개발자만의 재미 요소를 담아 높은 흥미를 끌 수 있도록 기획했습니다.
